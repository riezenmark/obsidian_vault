[[RACI]]

A - отвечает за активность
R - выполняет активность
C - консультирует/ помогает
I - уведомляется

|                      | Разработчик | Тестировщик | Тимлид |
| -------------------- | ----------- | ----------- | ------ |
| Проектирование       | I           |             | C      |
| Разработка           | R           | I           | ARC    |
| Тестирование         | C           | R           | AC     |
| Код-ревью            | IC          |             | AR     |
| Релиз                | I           | I           | R      |
| Общение с заказчиком |             |             |        |
